# TO-DO-on-React
> Это приложение для управления списком дел

## Используемые технологии:

- HTML
- CSS,SCSS
- TypeScript
- React, React-Router-Dom
- Redux(Toolkit)
- Material UI
- Сборщик Webpack

## Как посмотреть проект:

- git clone "ссылка на репозиторий"
- npm install
- npm run build
- npm start
